## üß† How It Works

The project is divided into **two main units**:


### ‚úã Hand Gesture Controller
- Reads hand tilt using a motion sensor

### üöÅ Mini Drone
- Receives commands from the controller

- Moves based on user gestures

## ‚úã Gesture Mapping

| Hand Gesture | Drone Action |
|-------------|-------------|
